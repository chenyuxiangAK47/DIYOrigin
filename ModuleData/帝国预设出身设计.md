# 帝国预设出身系统设计

## 历史背景参考

本设计参照：
- **东罗马帝国（拜占庭）**：军区制、元老院、教会、官僚体系
- **罗马三王并立时期**：君士坦丁、马克森提乌斯、李锡尼等皇帝并立，政治斗争激烈

## 设计理念

- **家族史叙事**：所有描述采用"家族记忆/历史传述"的口吻
- **历史负担**：每个出身都带有"旧仇/誓约/流亡/污名"等历史标签
- **三王并立背景**：所有出身都受到三王并立的政治环境影响

## 预设出身列表（8个）

### 1. 元老院流亡者 (senate_exile)

**描述**：你的家族曾是元老院的一员，但在三王并立的权力斗争中站错了队。如今你带着家族的名号与残余势力，在帝国的夹缝中寻求复兴。

**节点流程**：
1. **你的家族为何被流放**
   - 支持了败北的皇帝
   - 被指控贪腐与滥用职权
   - 参与过失败的政变
   - 外交政策失误导致边境失守

2. **你带走了什么家族遗产**
   - 元老院印章与家谱
   - 家族积累的财富
   - 忠诚的客户与门客
   - 政治文件与把柄

3. **你的随从是什么人**
   - 被释放的奴隶与自由民
   - 旧部军团老兵
   - 雇佣的佣兵

4. **你打算如何复兴家族**
   - 通过合法途径恢复地位
   - 寻找新的政治盟友
   - 用武力重新夺回权力

### 2. 边境军团叛将 (legion_deserter)

**描述**：你曾是帝国边境军团的一名军官，但在三王并立的混乱中，你选择了背叛。你带着旧部离开，在帝国的边缘地带寻求新的出路。

**节点流程**：
1. **你为何背叛军团**
   - 军饷被克扣，士兵不满
   - 拒绝为新皇帝效忠
   - 拒绝执行屠杀命令
   - 与上级有私仇

2. **你带走了哪些旧部**
   - 步兵老兵为主
   - 骑兵为主
   - 混编旧部

3. **你带走了什么军需**
   - 带走一批军械
   - 带走一袋军饷
   - 带走军团旗帜与文件

4. **你打算如何生存**
   - 当雇佣军
   - 当军阀
   - 离开帝国

### 3. 帝国贸易家族 (imperial_merchant)

**描述**：你的家族世代经营帝国商路，在三王并立的混乱中，你必须在政治漩涡中保护家族的贸易网络。你是商人，也是政治玩家。

**节点流程**：
1. **你的家族主要经营什么**
   - 粮食与补给
   - 奢侈品与香料
   - 武器与军需
   - 奴隶贸易

2. **你与哪个帝国派系有联系**
   - 与北部帝国联系紧密
   - 与南部帝国联系紧密
   - 与西部帝国联系紧密
   - 保持中立，谁都不站

3. **你的商队护卫是什么风格**
   - 雇佣退役军团士兵
   - 雇佣佣兵
   - 混编护卫

4. **你如何应对三王并立**
   - 只求利润，不站队
   - 通过贸易影响政治
   - 建立独立贸易网络

### 4. 被废黜的贵族 (deposed_noble)

   - 家族印章与家谱

   - 隐藏的财富
   - 忠诚的客户与门客

3. **你的随从是什么人**
   - 家族成员与亲信
   - 旧部老兵
   - 雇佣的佣兵

4. **你打算如何复仇**
   - 通过合法途径恢复地位

## 设计特点

1. **历史厚重感**：每个出身都带有历史背景和政治标签
2. **三王并立背景**：所有出身都受到三王并立的政治环境影响
3. **家族史叙事**：采用"家族记忆"的口吻，而非个人传记
4. **平衡设计**：强模板（兵多但关系差）vs 弱模板（兵少但关系好）
5. **历史负担**：旧仇、誓约、流亡、污名等标签影响后续事件

## 实现建议

1. **外交关系**：使用 `relation_delta:empire_north/south/west` 来区分三个帝国派系
2. **历史标签**：使用 `flags_add` 来标记历史负担（如 `flag_political_debt`, `flag_is_suspected`）
3. **兵力平衡**：遵循"强模板兵多但关系差，弱模板兵少但关系好"的原则
4. **叙事风格**：所有文本采用"家族史"口吻，而非个人传记
